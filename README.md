# k_filter - Embedded Filtering Library

Lightweight, dependency-free filter implementations for embedded systems. Currently supports:

- Moving Average Filter
- Low-Pass IIR Filter
- Median Filter
- Exponential Moving Average
- 1D Kalman Filter

## Usage

Include `k_filter.h` in your project and link with `k_filter.c`.

See `examples/filter_test.c` for a sample use case.
